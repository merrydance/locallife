## 三、软件功能与特点（500 字内参考文案）

> 以下文案可直接复制填入申请表"软件功能与特点"栏目，按实际情况微调。

---

乐客来福商户是面向乐客来福本地生活服务平台入驻商家的 Android 移动端管理工具。主要功能如下：

1. **账号绑定认证**：通过微信小程序生成的六位绑定码完成设备与商户账号的安全绑定，采用 JWT Token 双端认证体系，Token 加密存储于 Android Keystore 安全区。

2. **多通道订单推送**：同时维护 WebSocket 实时通道、各厂商原生推送通道（华为 HMS、小米 MiPush、OPPO Push、vivo Push）以及 30 秒定时轮询三路并行，确保在设备息屏或后台场景下亦能可靠接收新订单通知。

3. **消息去重机制**：基于 message_id 的 SQLite 本地去重库，防止三通道重复推送触发重复报警。

4. **订单管理**：订单列表按"待接单 / 进行中 / 已完成"三状态分栏展示，支持查看完整订单详情及操作接单。

5. **多感官告警**：新订单到达时通过全屏弹窗唤醒锁屏界面、播放预录提示音、TTS 语音播报订单号与金额，多维度提醒商家。

6. **蓝牙小票打印**：自动扫描并连接蓝牙热敏打印机，接单后自动打印商品明细小票，支持手动重打。

7. **设备保活**：内置 Android 前台常驻服务，配合主流品牌机型保活权限引导页，保证软件在各厂商定制 ROM 下持续运行。

8. **在线升级**：启动时检测最新版本，支持强制和非强制 APK 热更新。

## 八、常见注意事项

1. **软件名称须唯一清晰**：申请表中的软件名称与源程序、文档首页的名称须完全一致，不得出现"助手""系统""平台"等模糊词（除非确实如此命名）。

2. **源程序不得全部遮挡**：若含商业秘密需要遮挡，遮挡内容不得超过每页 50% 行数，且遮挡必须在打印前完成（黑色色块遮盖，扫描件不可用）。本项目源程序无须遮挡。

3. **开发完成日期须可信**：建议保留开发过程中的 Git commit 记录、项目管理系统截图、相关合同或发票等辅助文件备查，日后若发生著作权争议可作为佐证。

4. **在线申请更快**：在线提交资料后，只需邮寄纸质申请表（或使用电子签章全程在线），大幅缩短流程。

5. **电子版证书同等法律效力**：CCPC 出具的电子证书与纸质证书具有同等法律效力，满足投标、工商注册等常见使用场景。
